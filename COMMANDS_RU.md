>[!TIP]
> Здесь написаны все команды, которые можно использовать в консоли

## помощь по консоле
``` help ```

## для включения sql мода
``` sql ```
>[!TIP]
> в режиме sql вы можете писать не каждую новую строку, а сразу несколько. Вы должны написать это с самого начала ``` !one line ``` или ``` !morel ```, затем вы можете написать команды (максимум 100) для завершения ввода, написать ~ или готово (если лимит превышен, команда завершает ввод и выполняется

Команды для очистки консоли
``` clear ```
``` cls ```

удалить данные для быстрого входа в вашу базу данных
``` clear data ```

>[!TIP]
> команды хранятся в json файле

для создания вашей команды (для быстрого написания sql запросов)
```mkcommand (name command) (command)```

для удаление команды из json 
```rmc (your name command)```

для вывода всех ваших команд
```all-commands```
```output-all-commands```

чтобы запустить команду, вам нужно написать так
```run (your command)```

для выхода
``` exit ```
``` ex ```
